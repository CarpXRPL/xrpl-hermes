# TX Ecosystem — Complete Reference

## Overview
TX is the XRP Ledger ecosystem's multi-chain settlement protocol and NFT marketplace. TX connects XRPL L1, EVM sidechain, and other chains through a native bridge. Native token: TX.

## Core Products
- TX Marketplace: NFT marketplace supporting XLS-20 NFTs natively. Buy, sell, auction, creator royalties, rarity tracking.
- TX Bridge: Cross-chain bridge between XRPL L1, EVM sidechain, and other chains. TX token used for bridge fees. Federator-backed.
- TX Portal: Web dashboard for wallet connect (Xaman/MetaMask), portfolio tracking, NFT gallery.

## TX Token
- Currency: TX on XRPL (verify issuer from official sources)
- Uses: marketplace fees, bridge fees, staking, governance
- Cross-chain: ERC-20 on EVM sidechain
